Root Folder
yarn install

client folder
yarn install

Development

go to root folder, run
yarn run both

or terminal 1
yarn run dev

and terminal 2
cd client
yarn run start

For Server:
http://localhost:4000/
and for client:
http://localhost:3000/

Production:
client folder

yarn run build

root folder
yarn run build
yarn run prod

Go to http://localhost:4001/

Features:
Use of Typescript on Node js and React layer.
Server is on Node JS
Client is on React (in client folder)

    React:
        Use of BrowserRouting,
        Use of Material UI,
        Use of Axios
        Use of jotai local storage for caching data for 30 mins (we can change this based on requirement)
